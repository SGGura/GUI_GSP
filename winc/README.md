# WINC1500 OTA image

Файл `m2m_ota_3a0.bin` — образ для команды `wifi ota` на iEMAT1 (WINC1500 3A0).

Собран из `Scripts` после `flash_winc.bat` / `prepare_image.cmd` (прошивка 19.7.10).

## Прямая ссылка (для WINC, без редиректа, &lt; 100 символов)

```
https://raw.githubusercontent.com/SGGura/GUI_GSP/main/winc/m2m_ota_3a0.bin
```

## Команда на приборе

```
wifi ota GURAWORK GURA03071963 m2m_ota_3a0.bin
```

или полный URL:

```
wifi ota GURAWORK GURA03071963 https://raw.githubusercontent.com/SGGura/GUI_GSP/main/winc/m2m_ota_3a0.bin
```
