# T-Rex Dino Game (Chrome-style) for GUI_GSP

Клон игры с динозавриком из Chrome (chrome://dino) для монохромного дисплея
128×64, написанный на C с использованием библиотеки GUI_GSP.

## Структура

| Файл | Описание |
|------|----------|
| `dino_game.h` | Заголовок — типы, API игры |
| `dino_game.c` | Вся игровая логика (платформо-независимая) |
| `main_sdl.c`  | SDL2 симулятор для ПК (для отладки и демо) |
| `main_gsp.c`  | Интеграция с GUI_GSP (для embedded-устройства) |
| `Makefile`     | Сборка SDL2 версии |

## Сборка SDL2 симулятора (ПК)

```bash
sudo apt install libsdl2-dev   # установить SDL2
cd Examples/dino_game
make
./dino_sdl
```

## Управление

| Клавиша | Действие |
|---------|----------|
| `SPACE` / `↑` | Прыжок / Рестарт после Game Over |
| `↓` | Пригнуться |
| `ESC` / `Q` | Выход |

## Интеграция с GUI_GSP (embedded)

1. Добавьте `dino_game.c`, `dino_game.h` и `main_gsp.c` в проект.
2. Настройте `GSP_HW` (дисплей, клавиатура) в `GSP_Hardware.c`.
3. В прерывании таймера вызывайте `GUI_TimerClock(period_ms)`.
4. Задайте `DINO_KEY_JUMP` и `DINO_KEY_DUCK` под ваши кнопки.

## Особенности

- **128×64 1-bit** — оптимизирован под маленький монохромный дисплей
- **Ночной режим** — переключается автоматически каждые ~23 секунды
- **Кактусы** трёх видов + **птеродактиль**
- **Хай-скор** сохраняется между раундами
- **Прыжок** и **пригибание** с хитбокс-проверкой
- **Портативная логика** — вся игра в `dino_game.c`, без зависимостей
